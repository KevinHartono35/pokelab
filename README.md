# Endpoints

The API for pokemon data are listed below:

- List of pokemon https://pokeapi.co/api/v2/pokemon

- Detail of pokemon https://pokeapi.co/api/v2/pokemon/{name or id}

https://pokeapi.co/docs/v2#pokemon-section
